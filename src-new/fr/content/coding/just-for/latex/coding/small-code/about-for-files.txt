===========================================
path::''about.yaml'' pour les fichiers \src
===========================================

Dans chaque sous-dossier \src peut être mis un fichier path::''about.yaml'' permettant de choisir l'ordre d'analyse des fichiers.
Dans cette section, nous allons nous concentrer sur les fichiers ext::''sty'' liés au code de notre exemple, mais soulignons que l'on peut aussi, si besoin, indiquer des fichiers ext::''cls'' (nous reviendrons sur les fichiers ext::''tex'' pour la \doc "utilisateur" dans la section suivante).
Imaginons que l'on ait le code suivant pour notre exemple ficitf.

yaml::
    ---
    title = Le fichier path::''src/poor-lonesome-cowboy/about.yaml''
    ---

    toc:
      - rantanplan.sty
      - lucky.sty


Le code précédent sert à considérer le fichier path::''rantanplan.sty'' avant path::''lucky.sty'' et non en suivant l'ordre \lexico qui est le comportement par défaut.
