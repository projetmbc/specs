==============================
Noms des macros "traductrices"
==============================

Les noms sont \tjrs construits suivant le même schéma. Considérons par exemple le package \tutodoc daté du \tutodocdate.

    1) verb::''exa_title = Exemple'' est présent dans un fichier de traductions pour le français.

    1) Dans le fichier path::''french.sty'' est ajoutée "\auto^t" la macro ''\tutodoc@trans@exa@title'' où le schéma de nommage est du type ''\monpackage@trans@id@une@trad'' où verb::''id_une_trad'' est un identifiant utilisé dans les fichiers ext::''txt'' de traductions.


note::
    Les outils automatisés doivent gérer la présence éventuelle de doublons ! L'auteur de ce \docu utilise \texfacto.
