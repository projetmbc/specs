=============================
Montrer des cas d'utilisation
=============================

Le package \bdoc permet de passer via des \envs comme ''\begin{bdoclatex} ... \end{bdoclatex}'' présenté ci-dessous, ou des macros pour importer des contenus de fichiers externes.

latex::
    \begin{bdoclatex}
      $\norm{\vect{i}} = \vnorm{i}$
    \end{bdoclatex}
