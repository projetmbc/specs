====================
Couleurs... ou pas !
====================

Par défaut, les fonctionnalités proposées sont toutes faites pour un mode noir-et-blanc
((
    Utile pour une impression de \docus.
)).
Il est tout de même possible d'obtenir facilement un rendu coloré
((
    Utile pour une lecture sur écran.
)).
Ceci se fait via une option de \pack demandant d'utiliser l'un des thèmes proposés, le noir-et-blanc étant celui par défaut.


warning::
    Les versions colorées et celle noir-et-blanc peuvent différées dans leur \mef : on ne fera pas juste un passage à un niveau de gris car ceci est rarement pertinent pour des \docus destinés à être photocopiés.


note::
    En coulisse, les \mefs sont obtenues via les outils fournis par le \pack \tnsformat.
